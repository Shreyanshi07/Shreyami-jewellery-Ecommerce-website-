# Shreyami Jewellery - -Commerce UI

A minimalist, high-end e-commerce web application designed for a luxury jewelry brand. This project focuses on a seamless user experience, responsive design, and dynamic client-side logic.

## Key Features
- **Dynamic Product Filtering:** Browse collections categorized by 'Office Wear', 'Daily Wear', and 'Party Wear'.
- **Smart Cart System:** Fully functional 'Add to Cart' and 'Remove' logic using **JavaScript LocalStorage** for data persistence.
- **Personalized User Dashboard:** Features a dynamic user profile that automatically generates avatars and greetings based on login data.
- **Luxury UI/UX:** Built with a "Mobile-First" approach, featuring custom Lightbox galleries and smooth CSS transitions.
- **Authentication Flow:** Integrated login/signup simulation to gate-keep the shopping experience.

## Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Grid & Flexbox)
- **Scripting:** Vanilla JavaScript (ES6+)
- **Storage:** Web Storage API (LocalStorage)
- **Icons:** FontAwesome & Google Fonts
